# Public Site Assets

This folder contains the static landing-page assets for xLedgRSv2Beta.

- `index.html` - Static project landing page.
- `xLedgRSv2Beta-mark.svg` / `.png` - Project mark assets.
- `xLedgRSv2Beta-og.svg` / `.png` - Open Graph/social preview assets.

These files are presentation assets only; they are not used by the node runtime.
